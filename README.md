# アプリケーション名
**まとめて開くんです**

## 概要
実行ファイルやサイトのURLなどを一括で開くアプリケーションです。
内容が同じ複数のファイルやURLを何度も開くことがあったため、一括で楽に開くことが出来るようなアプリケーションが欲しく、作成しました。

## 必須要件
- WindowsOS

## 使い方
1. ReleaseからZIPファイルをダウンロードします。
2. ZIPファイルを解凍し、setup.exeを起動します。
3. インストーラーが起動されるので、手順に従って進めてください。

## 機能
### ショートカット一覧
ショートカットの一覧を確認することが出来、「新規作成・編集・デスクトップ上に生成・削除」の4つを行えます。
### 新規・編集
ショートカットの新規作成・編集ができます。
